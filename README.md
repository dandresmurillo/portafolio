## 🚀 Despliegue en GitHub Pages

Este sitio está desplegado automáticamente con GitHub Pages desde la rama `main`.

Para activarlo:
1. Ir a Settings → Pages en el repositorio
2. Seleccionar Source: **Deploy from a branch**
3. Branch: `main` / `/ (root)`
4. Guardar
